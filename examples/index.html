Introducing Moveit Library

Moveit Library is a lightweight SVG path manipulation and animation library

How to use

1. Include the link to Moveit library javascript file right before </body> tag:

<script src="https://kodhus.com/moveit-0.5.js"></script>

2. Create an svg path in your html and add the following code to your javascript:

const yourElement = document.querySelector(".your-element-class");
const moveit = new Moveit(yourElement, {
  start: "0%",
  end: "0%",
});
moveit.set({
  start: "0%",
  end: "100",
  duration: 2,
});

Inline documentation

Setting Start and end of the visible portion of the full svg path:

const moveit = new Moveit('YOUR SVG PATH SELECTOR', {
  start: 'start perentage of the path%',
  end: 'end percentage of the path%',
});

Animating SVG path:

moveit.set('YOUR SVG PATH SELECTOR', {
  start: 'start perentage of the path%%',
  end: 'end percentage of the path%',
  duration: [Duration in second(s)],
  delay: [Starting delay of animation in second(s)],
  timing: 'ease-in' | 'ease-out' | 'ease-in-out' | 'cubic-bezier'
});
